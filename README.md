# Awesome-chocolates-sales

This Power BI project analyzes sales performance data for a fictional company, Awesome Chocolate Sales.

The dashboard provides a comprehensive view of revenue, profit, product performance, regional sales trends, and sales team contributions. It enables business stakeholders to monitor KPIs, identify growth opportunities, and make data-driven decisions.
🎯 Business Objectives

Analyze total sales and profit performance

Identify top-performing products

Compare regional sales trends

Evaluate sales representative performance
🗂️ Dataset Description

The dataset includes:

Order ID

Order Date

Product Name

Category

Sales Amount

Cost

Profit

Region / Country

Sales Representative

Quantity Sold

⚙️ Tools & Technologies Used

Power BI Desktop

Power Query – Data cleaning and transformation

DAX – Custom calculations and KPIs

Data Modeling (Star Schema approach
